Overview

Guilds

  • Guilds are one of the newly tracked Sea 2 systems.
  • The current Guilds menu confirms creating a guild requires Level 10,000 and 1 Guild Key.
  • The same menu confirms the create cost is 1 Guild Key and joining uses ID search or invitations.
  • Guild Key is now tracked on the current Sea 2 rows at 0.00019342% on Delinquent, 0.00019742% on Strong Fighter, 0.00024742% on Strong Bandit, 0.00026342% on Fast Ninja, 0.011% on Sea Serpent and Kraken, and 0.01% on Cosmic Being Boss.
  • The newer current Guild upgrade board confirms six tracked guild-upgrade lines so far: Member Capacity, Damage %, Crit Damage, Crit Chance, HP %, and Luck.
  • The same upgrade board shows Damage, Crit Damage, Crit Chance, HP, and Luck each cap at Level 5, while Member Capacity caps at Level 10.
  • Players searching Guild Key dropchance or Guild Key drop rate can use this page for the current Sea 2 source list and the best published Guild Key farming routes.

How guilds work right now

  • Use the Guilds menu to create a guild, join a guild by ID, or check invitations.
  • Creating a guild currently needs Level 10,000 and 1 Guild Key.
  • Guild Key is now tracked on the current Sea 2 routes at 0.00019342% on Delinquent, 0.00019742% on Strong Fighter, 0.00024742% on Strong Bandit, 0.00026342% on Fast Ninja, 0.011% on Sea Serpent and Kraken, and 0.01% on Cosmic Being Boss.
  • The visible Guilds menu also shows Top Guilds and Rewards tabs, which means the system already tracks leaderboard and reward flow even though the details are not fully captured yet.

Guild upgrade notes

  • Damage, Crit Damage, Crit Chance, HP, and Luck all currently look like 5-level guild upgrades on the current board.
  • Member Capacity currently looks like the only tracked 10-level guild upgrade on the current board.
  • The current best-luck build page already uses +10% Guild Luck as the max tracked guild contribution, which matches the current Luck board at 5 levels of +2% each.
  • The currently confirmed costs are Damage 50, Crit Damage 35, Crit Chance 35, HP 35, Luck 65, and Member Capacity Lv. 2 at 150 from the visible upgrade board.
  • Member Capacity is the odd one out because the captured button shows Lv. 1 to Lv. 2 rather than Lv. 0 to Lv. 1, so Lv. 1 is still left as TBD until another screenshot settles it.

Guilds FAQ
What is the Guild Key drop chance in Sailor Piece?
The current tracked Guild Key drop chance starts at 0.00019342% on Delinquent, rises through the late Sea 2 normal-enemy routes, reaches 0.01% on Cosmic Being Boss, and tops out at 0.011% on Sea Serpent and Kraken.
Which NPC or boss has the best Guild Key drop chance right now?
Sea Serpent and Kraken currently share the strongest published Guild Key line at 0.011%, while Fast Ninja is the best normal-enemy Guild Key farm at 0.00026342%.
