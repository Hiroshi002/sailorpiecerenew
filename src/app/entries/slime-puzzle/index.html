Slime Puzzle

Seven hidden slime collectibles tied to the Slime Key Crafter on Slime Island and the route into the Slime Crafter unlock.

Systems Verified
Total Slimes 7
Main NPC Slime Key Crafter
Main island Slime Island
Main use Unlock the Slime Crafter route
Overview

Slime Puzzle

  • The Slime Puzzle is a hidden collection route tied to the Slime Key Crafter on Slime Island.
  • Players need all 7 Slimes before they can finish the Slime Crafter route.
  • The puzzle overlaps naturally with Skill Tree progression because both routes sit on Slime Island.
Section

How the Slime Puzzle works

  • Travel to Slime Island and find the Slime Key Crafter NPC.
  • Collect all 7 hidden Slimes placed across several islands.
  • Return to the Slime Key Crafter after the full set is complete to continue the Slime Crafter route.

Section

Why players do the Slime route

  • It unlocks the Slime Crafter line instead of acting like a normal combat reward.
  • It overlaps well with Skill Tree progression because players already revisit Slime Island for late-game stat scaling.
  • It is one of the cleaner checklist systems in Sailor Piece because each Slime is a fixed collectible rather than a random drop.
