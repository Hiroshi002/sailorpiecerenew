Overview

True Aizen / True Manipulator

  • True Aizen needs the base Manipulator Sword before you can buy it.
  • Current item guides also describe the same upgrade target as the True Manipulator sword, while sword lists still commonly use the name True Aizen.
  • The purchase bundle is 35,000 Gems, 75 Chrysalis Sigil, 15 Fusion Ring, 8 Divinity Essence, 3 Transcendent Core, 1 Evolution Fragment, the Transcendent Being title, and the base Aizen or Manipulator Sword.
Section

How to obtain True Aizen

  • Own the base Manipulator Sword first.
  • Unlock Soul Dominion by collecting 6 Hogyoku Fragments and finishing the Hollow Island (Hueco Mundo) path.
  • Travel to Soul Dominion and talk to the True Aizen NPC.
  • Bring the full gem, title, and boss-drop bundle.
