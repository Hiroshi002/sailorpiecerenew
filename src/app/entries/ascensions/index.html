Overview

Ascensions

Page Filter

Useful for big overview pages when you want one item fast.

  • Ascension is a late-game progression system handled by the Ascension NPC on Sailor Island.
  • Current community guides agree that ascension stacks permanent boosts to money, gems, cooldown reduction, and luck.
  • Some recent code waves are also reported by players to use ascension locks, but current public code trackers do not publish those gates consistently.
Section

How to ascend

  • Travel to Sailor Island and talk to the Ascension NPC inside the temple area.
  • Complete the listed level, kill, damage, ability, dungeon, and material requirements for your next ascension tier.
  • When every requirement is green, interact with the NPC again and ascend to claim the permanent stat buffs.

Fast notes before you start the grind

  • Ascension 6 and higher start mixing Rune Dungeon progress into the requirement list, so rune farming and ascension farming overlap well.
  • Ascension 9 and 10 pull in Boss Rush and Infinite Tower goals, so they act as true endgame checks instead of simple level gates.
  • If a code fails to redeem, check your level first. Some newer codes are also reported by players to use ascension gates, but public code pages do not list those gates consistently yet.
