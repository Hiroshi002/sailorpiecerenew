## 🛠️ INSTALLATION & DEVELOPMENT
> [!IMPORTANT]
> โปรเจกต์นี้ต้องการ **Node.js 18+** เพื่อประสิทธิภาพสูงสุดในการรัน Web Dashboard
```bash
# Clone the treasure map
git clone [https://github.com/your-username/sailor-piece.git](https://github.com/your-username/sailor-piece.git)
# Recruit the crew (Install dependencies)
npm install
# Set sail (Start development)
npm run dev
